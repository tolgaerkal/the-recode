# THE RECODE
### 32 Sequences to rewire how you think, build, and lead in the age of models.

> *You are not broken. You are running outdated code.*

---

## What is The Recode?

**The Recode** is a book concept and interactive web experience built around a single premise: the shift to a model-native world is not a technology problem — it is a thinking problem.

Most people try to layer new tools on top of old mental models. The Recode does something different. It proposes a complete rewrite — in four phases:

| Phase | What it does |
|-------|-------------|
| `// DELETE` (seq 01–08) | Uninstalls the assumptions that no longer serve you |
| `// DEBUG` (seq 09–16) | Finds where your current thinking is broken |
| `// RECODE` (seq 17–24) | Installs a new operating system for thinking and building |
| `// RUN` (seq 25–32) | Puts the new code into practice — as a leader, learner, and human |

---

## The Experience

The interactive version of The Recode is a single-page HTML experience — designed to feel like a book that lives on screen. Blueprint-style SVG diagrams. Typographic rigour. Every sequence is a standalone insight, and together they form a complete system.

No build step required. Just open `index.html` in a browser.

```bash

cd the-recode
open index.html
```
